Горячие клавиши:
Интерфейс личной рации - lCtrl + P;
Интерфейс дальней рации - lAlt + P;
Интерфейс переговорного устройства дайверов - Shift + P;
Разговор по личной рации - Caps Lock;
Разговор по дальней рации - lCtrl + Caps Lock;
Разговор по переговорному устройству для дайверов - Ctrl + ~;
Передача на доп. канале личной рации - Т;
Передача на доп. канале дальней рации - Y;
Переключение каналов рации - NUM 1-9;
Переключение дальней - lAlt + NUM 1-9;
Переключение стерео КВ - lCtrl + Стрелочки(Лево, Право, Вниз);
Переключение стерео ДВ - lAlt + Стрелочки(Лево, Право, Вниз);
